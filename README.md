# 🦇 Bat Signal App

Aplicativo desenvolvido em **React Native (Expo)** que simula o envio de um pedido de ajuda usando o icônico **Bat-Sinal** do universo do Batman.

---

## 🚀 Funcionalidades

✔ Tela inicial com ativação do Bat-Sinal
✔ Formulário de solicitação de ajuda
✔ Validação de campos obrigatórios
✔ Feedback visual com alerta
✔ Fechamento automático do teclado
✔ Navegação entre telas (sem biblioteca externa)

---

## 📱 Preview

> Interface simples, escura e inspirada no tema do Batman 🖤💛

* Tela 1: Botão **"Activate Bat Signal"**
* Tela 2: Formulário com:

  * Nome
  * Telefone
  * Localização
  * Observação

## 📦 Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/bat-signal-app.git
```

### 2. Acesse a pasta

```bash
cd bat-signal-app
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Inicie o projeto

```bash
npx expo start
```
